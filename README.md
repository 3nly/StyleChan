====

StyleChan is a userscript allowing you to customize 4chan with various themes and features.\
Works with both **native 4chan** and **4chan X**.

Originally developed by [seaweedchan](https://github.com/seaweedchan/OneeChan), this is a fork of OneeChan.

## [Click here to Install](https://github.com/3nly/StyleChan/raw/refs/heads/main/builds/StyleChan.user.js)

 
 **Minified Version** → [StyleChan.min.user.js](https://github.com/3nly/StyleChan/raw/refs/heads/main/builds/StyleChan.min.user.js) 

- Firefox: requires [Violentmonkey](https://addons.mozilla.org/en-US/firefox/addon/violentmonkey/)
- Chrome/Edge: requires [Violentmonkey](https://chromewebstore.google.com/detail/violentmonkey/jinjaccalgkegednnccohejagnlnfdag/)

Get [more themes](https://github.com/3nly/StyleChan/wiki/Custom-Themes#custom-themes).

## Troubleshooting

- If you have problems, try first resetting your StyleChan settings (**CTRL + F1 → Reset**) and restarting your browser.
  
- Check if it's a 4chan or 4chan X issue - StyleChan primarily deals with stylesheets.
  
- Report issues on the [Issues](https://github.com/3nly/StyleChan/issues) page.
